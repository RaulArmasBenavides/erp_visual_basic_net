# ERP Database en VB.NET

Este repositorio contiene el código fuente
de una base de datos ERP implementada en VB.NET utilizando el .NET Framework 4.5.

## Características
- **Base de Datos ERP**: Gestión integral de los recursos empresariales.
- **Configuraciones de Comunicación**: Integración y configuración para plataformas de comunicación como Zoom y Google Teams.
- **Configuraciones de Seguridad**: Reglas y configuraciones para firewall y otros aspectos de seguridad.
- **Capa de Instalador**: Herramienta que facilita la instalación y despliegue de la aplicación.

## Requisitos

- Microsoft .NET Framework 4.5
- Acceso administrativo para configuraciones de firewall.

## Instalación

1. Clona este repositorio en tu máquina local.
2. Navega al directorio del proyecto y busca la capa de instalador.
3. Ejecuta el instalador y sigue las instrucciones en pantalla.

## Configuración

### Zoom y Google Teams

1. Abre la aplicación y navega a la sección de configuraciones.
2. Selecciona la pestaña de comunicaciones y sigue las instrucciones para integrar Zoom y/o Google Teams.

### Firewall

1. Asegúrate de tener derechos de administrador.
2. Navega a la sección de configuraciones de seguridad.
3. Sigue las instrucciones para establecer las reglas y configuraciones del firewall.
